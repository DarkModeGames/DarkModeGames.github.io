Recipes

Reusable patterns for building campaigns and plugins with Isometry data. Each recipe states a Problem, the Pattern that solves it, and a worked YAML example.

Adding a recipe: drop a new web/recipes/<slug>.md file (the filename is the URL slug, the first # heading is the title), then add one bullet to the list below. Nothing else to wire up.
