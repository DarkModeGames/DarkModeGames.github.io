Introduction to Isometry

Isometry is an isometric pixel-art RPG framework. It provides a complete, data-driven system for creating custom campaigns with multiplayer support, AI behaviors, and rich gameplay mechanics.

What is Isometry?

Isometry is one app that does two things: it lets creators build games, and it lets players play them. Creators design worlds by writing YAML or JSON files — no programming required — and players load those worlds and play, solo or together over the network.

Key Features

  • Isometric grid-based maps — isometric maps with layers, terrain, and pathfinding.
  • Multiplayer networking — host, join, or run dedicated servers over the network.
  • Data-driven campaigns — define everything in YAML or JSON, no code required.
  • AI behavior system — goal-based NPC strategies and patrol routes.
  • Skill & action system — up to 9 hotbar skills, composed from 75 built-in action functions with conditional logic.
  • Resource management — health, mana, gold, and custom resources with triggers.
  • Vision & perception — line-of-sight, fog of war, and stealth mechanics.
  • Gamepad support — full controller mapping alongside remappable keyboard binds.

The whole game is described with a small vocabulary of 33 entity types.

System Requirements

  • OS: Linux (x86_64), Windows (x86_64), or macOS
  • Disk: ~200 MB for the engine, plus campaign data
  • RAM: 512 MB minimum
  • Network: Internet connection for multiplayer (UDP port 5000 by default)

Where to next

  • Installation — download Isometry and set it up with your campaigns.
  • Playing — controls, joining a campaign, and keybinds.
  • Hosting — run a host or dedicated server.
