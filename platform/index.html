Download Isometry

Isometry is a single executable — the engine is the product. Isometry is currently in open beta (v0.387).

Get it on itch.io

Isometry is available from itch.io:

Download on itch.io

The download includes builds for Linux, Windows, and macOS.

Running it

Put the Isometry executable in a folder, place your campaign .zip files alongside it, and launch:

  • Linux — mark the binary executable, then run it.
  • Windows — run the .exe (double-click, or launch from a terminal to pass flags for hosting and dedicated servers).
  • macOS — if Gatekeeper quarantines the app, clear it with xattr -dr com.apple.quarantine Isometry.app, then open it.
